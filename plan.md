# 충남교육청 고등학교 전입학 안내 웹페이지 — Claude Code 구축 계획

---

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 목적 | 충청남도교육청 누리집 내 고등학교 전입학 안내 게시판을 독립 HTML로 구축 |
| 대상 | 타시도·관내 전입학을 문의하는 학부모 및 학생 |
| 기술 스택 | 순수 HTML + CSS + Vanilla JS (외부 라이브러리 최소화) |
| 배포 | 충남교육청 누리집 내 탑재 또는 Netlify/GitHub Pages |

---

## 2. 페이지 구조

```
index.html                  ← 메인 (탭 레이아웃)
├── [탭 1] 고등학교 안내      ← 전입학 대상, 절차 요약, 문의처
├── [탭 2] 학교 소개 자료     ← 카카오맵 + 충남 고교 현황 (학교알리미 연계)
├── [탭 3] 고입전형 일정      ← 연도별 전입학 일정표
├── [탭 4] 고입전형의 이해    ← FAQ (자주 묻는 질문 아코디언)
└── [탭 5] 서식 자료실        ← 전입학 관련 서식 다운로드 목록

survey.html                 ← 전입학 가능 여부 모의 진단 설문 (별도 페이지)
```

## 3. 각 탭 상세 요구사항

### 탭 1 — 고등학교 안내
- 전입학 대상 안내 (일반계 / 특성화고 / 과학고·외고 등 계열별)
- 전입학 절차 단계 시각화 (신청→확인→제출→배정)
- 담당자 연락처 및 문의처
- 상단에 "전입학 가능 여부 진단하기" 버튼 → `survey.html` 링크

### 탭 2 — 학교 소개 자료
- 카카오맵 API 기반 충남 고등학교 분포 지도
- 시·군별 필터 (천안 / 아산 / 공주 / 기타 등)
- 학교 마커 클릭 시 학교명, 유형, 학교알리미 링크 표시
- 데이터: `schools.js` 별도 파일로 관리 (학교명, 좌표, 유형, URL)

### 탭 3 — 전입학 일정
- 연도별 전입학 가능 시기 표 (1학년 1학기 ~ 2학년 1학기)
- 학교 계열별 전입 가능 시기 표시
- 주요 일정 타임라인 컴포넌트

### 탭 4 — 고입전형의 이해 (FAQ)
- 아코디언 형식 FAQ 목록
- 예시 질문:
  - 전 가족이 이사해야 하나요?
  - 세대주 조건이란 무엇인가요?
  - 일반고와 특성화고 전입학 차이는?
  - 충남 내 같은 시 안에서 전학이 가능한가요?

### 탭 5 — 서식 자료실
- 서식 목록 테이블 (서식명 / 대상 / 파일 형식 / 다운로드 버튼)
- 예시 서식: 전·편입학 원서, 재학증명서 안내, 학교생활기록부 발급 안내

function getResult(answers) {
  const [q1,q2,q3,q4,q5,q6,q7] = answers;
  if (q1 === '아니오') return 'T1';
  if (q2 === '아니오') return 'T2';
  if (q7 === '과학고·외국어고·국제고·영재고·자율형 사립고') return 'T3';

  const school = q3, dest = q5, middle = q6;

  if (school === '충남 외 타시도') {
    if (dest === '아산') return middle === '아산' ? 'T12' : 'T14';
    if (dest === '천안') return middle === '천안' ? 'T13' : 'T15';
    if (dest === '충남 내 타시군(천안·아산 제외)') return 'T0';
    if (dest === '충남 외 타시도') return 'T11';
  }
  if (school === '충남 내 타시군(천안·아산 제외)') {
    if (dest === '아산') return middle === '아산' ? 'T4' : 'T14';
    if (dest === '천안') return middle === '천안' ? 'T5' : 'T15';
    if (dest === '충남 외 타시도') return 'T11';
    return 'T10';
  }
  if (school === '아산') {
    if (dest === '아산') return 'T9';
    if (dest === '천안') return middle === '아산' ? 'T6' : 'T16';
    if (dest === '충남 내 타시군(천안·아산 제외)' || dest === '충남 외 타시도') return 'T8';
  }
  if (school === '천안') {
    if (dest === '천안') return 'T9';
    if (dest === '아산') return middle === '천안' ? 'T7' : 'T16';
    if (dest === '충남 내 타시군(천안·아산 제외)' || dest === '충남 외 타시도') return 'T17';
  }
  return 'T0';
}
